# EduRegister — Student Course Registration System 🎓

EduRegister is a full-stack web application designed to digitize and simplify the college course registration process. It provides a user-friendly, responsive interface for students to securely log in, browse real-time course catalogs, and manage their academic enrollments online.

## 📖 Problem Statement
Traditional college course registration systems often rely on manual paperwork or outdated software, leading to administrative bottlenecks and a poor student experience. This project addresses the need for a centralized, digital solution where students can seamlessly create accounts, view available courses, and enroll dynamically. 

## ✨ Key Features
* **Authentication:** Secure student registration and login portals.
* **Interactive Dashboard:** A dynamic overview of active courses, credit hours, and enrollment statuses.
* **Course Catalog:** Browse available courses with details on credits, departments, and seat availability.
* **Enrollment Management:** Real-time adding and dropping of courses using asynchronous API calls.
* **Profile Management:** View and update personal and academic student information.

## 🛠️ Technology Stack
This application is built using a modern JavaScript-based architecture:

**Frontend (Client-Side)**
* **HTML5 & CSS3:** For a semantic, responsive, and accessible user interface.
* **JavaScript (ES6+):** Client-side logic, robust form validation, and user interactions.
* **DOM API:** For dynamic, seamless content updates without page reloads.
* **Fetch API:** For asynchronous data exchange (JSON) with the backend.

**Backend (Server-Side)**
* **Node.js:** JavaScript runtime environment.
* **Express.js:** Web framework for routing, API creation, and handling HTTP requests/responses.

**Database**
* **MongoDB:** NoSQL database supporting full CRUD operations for managing students, courses, and enrollments.

**Development Tools**
* **Version Control:** Git & GitHub
* **IDE:** Visual Studio Code (VS Code)
* **Debugging:** Browser DevTools
